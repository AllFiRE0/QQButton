## 📋 Возможности
- ✅ Динамическая кнопка в меню ESC (Server Links)
- ✅ Проверка условий (AND / OR)
- ✅ Поддержка PlaceholderAPI
- ✅ Выполнение команд при клике
- ✅ Все форматы цветов (&, §, MiniMessage, градиенты)

## 🔧 Команды
| Команда | Описание | Право |
| :--- | :--- | :--- |
| `/qqbutton reload` | Перезагрузить конфиг | `qqbutton.admin` |

## 📊 Права
| Право | Описание |
| :--- | :--- |
| `qqbutton.use` | Использование кнопки |
| `qqbutton.admin` | Доступ к админ-командам |

## 🎨 Цвета

Поддерживаются все форматы:
- `&f` - Legacy
- `§f` - Legacy (конвертируется)
- `<color:#FFFFFF>` - MiniMessage
- `<gradient:#FFFFFF:#000000>` - Градиент
- `{#FFFFFF>}текст{#000000<}` - Градиент
